# 🚀 C-DAC Training Projects

This repository contains the projects I developed during the **C-DAC Training Program**. These projects focus on Artificial Intelligence, Retrieval-Augmented Generation (RAG), Large Language Models (LLMs), and Deep Learning.

---

## 📂 Projects

### 🩺 AI Medico Bot
An AI-powered medical assistant that analyzes PDF medical reports using Retrieval-Augmented Generation (RAG) with LangChain, Google Gemini, and FAISS.

**Technologies:** Python, Streamlit, LangChain, Google Gemini, HuggingFace Embeddings, FAISS

📁 Folder: `AI_Medico_Bot`

---

### 📚 Mini RAG Studio
A lightweight RAG application that enables users to upload PDF documents and interact with them using Google's Gemini LLM.

**Technologies:** Python, Streamlit, LangChain, Google Gemini, FAISS

📁 Folder: `Mini_RAG_Studio`

---

### 🫁 Pneumonia Analysis
A deep learning project that detects pneumonia from chest X-ray images using Transfer Learning with the VGG16 architecture. Developed and trained using Google Colab.

**Technologies:** Python, TensorFlow, Keras, VGG16, OpenCV, NumPy, Matplotlib

📁 Folder: `Pneumonia_Analysis`
